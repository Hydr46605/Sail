# Sail 0.1.0-alpha.1

First public alpha of Sail Generic.

This release is intended for development, local integration testing, and
deployment staging. It is not a production authority.

## Included

- Sail Registry alpha service and OpenAPI contract.
- Velocity gateway plugin for Sail login enforcement.
- Paper companion plugin for identity diagnostics.
- Console alpha static build.
- Protocol metadata and release manifest.
- SHA-256 checksums for the release bundle and published files.

## Alpha Notes

- Use non-dev signing keys for persistent or public registries.
- Keep Velocity as the authentication boundary.
- Treat the Paper companion as diagnostics and metadata only.
- Review `SECURITY.md` before exposing a registry outside local development.
