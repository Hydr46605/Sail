# Sail 0.1.0-alpha.2

Second public alpha of Sail.

This release refreshes the alpha artifacts after the Global registry staging
pass. It is intended for development, local integration testing, and deployment
staging. It is not a production authority.

## Included

- Sail Registry production runtime entrypoint.
- Velocity gateway default profile pointed at Sail Global.
- Updated Gateway and Companion plugin artifacts.
- Console alpha static build.
- Protocol metadata, OpenAPI contract, release manifest, and SHA-256 checksums.

## Alpha Notes

- Use non-dev signing keys for persistent or public registries.
- Keep Velocity as the authentication boundary.
- Treat the Paper companion as diagnostics and metadata only.
- Review `SECURITY.md` before exposing a registry outside local development.
