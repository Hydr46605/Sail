# Sail 0.1.0-alpha.3

Third public alpha of Sail Generic.

This release updates the Sail Global default endpoint to the working public
registry hostname.

## Included

- Velocity gateway defaults now point at `https://sail.creepers.sbs`.
- Updated alpha release manifest and plugin artifacts.
- Console alpha static build.
- Protocol metadata, OpenAPI contract, release manifest, and SHA-256 checksums.

## Alpha Notes

- Use non-dev signing keys for persistent or public registries.
- Keep Velocity as the authentication boundary.
- Treat the Paper companion as diagnostics and metadata only.
