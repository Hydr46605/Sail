# Sail 0.1.0-alpha.4

This alpha focuses on repository hygiene, release correctness, and honest
Minecraft gateway behavior.

## Changes

- Blocks local-only files, generated clutter, and common secret formats from
  public repository checks.
- Keeps alpha GitHub releases marked as prereleases.
- Documents Java 21 as the release build runtime for Minecraft artifacts.
- Keeps the current gateway runtime focused on kick-mode authentication while
  limbo and hybrid remain future real implementations.
- Keeps the Paper companion positioned as diagnostics-only.
