# Sail 0.1.0-alpha.6

This alpha tightens the gateway and companion runtime after the first
limbo/hybrid implementation pass.

## Changes

- Hardens gateway configuration validation before the Velocity plugin starts.
- Tightens LimboAPI-backed runtime checks for `limbo` and `hybrid` modes.
- Clears malformed login challenges instead of leaving confusing retry state.
- Adds a console operator summary for session and trusted-server coverage.
- Tightens console session handling and callback cleanup.
- Clarifies Paper companion operator status and diagnostics.
- Strengthens local smoke checks around release bundle output and Minecraft
  plugin artifacts.
